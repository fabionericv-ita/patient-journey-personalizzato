# Patient Journey Portal - Frontend (React)

Interfaccia utente interattiva e moderna per il progetto **Patient Journey**, focalizzata sulla User Experience del paziente nel monitoraggio del proprio percorso di cura.

## 🎨 User Experience & UI
Il portale offre una navigazione fluida e intuitiva, progettata per garantire chiarezza in contesti medici complessi.

### Caratteristiche principali:
* **Timeline Interattiva:** Visualizzazione dinamica degli appuntamenti passati e futuri.
* **Preparazione Esami:** Modulo informativo contestuale (es. istruzioni "24h prima") basato sul tipo di prestazione.
* **Gestione Documentale:** Interfaccia per il download sicuro dei referti digitali.
* **Design Professionale:** Coerenza visiva ottenuta tramite **Material Icons** e una palette cromatica istituzionale (Background: #616a97).

## 🛠️ Stack Tecnologico
* **Framework:** React.js
* **Iconografia:** Material Icons
* **Styling:** CSS3 (con gestione rigorosa del layout e delle varianti dei form)
* **Deploy:** Vercel

---
*Design focalizzato su accessibilità e chiarezza del dato sanitario.*
